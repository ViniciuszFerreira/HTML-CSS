O HTML (HTML) é uma linguagem de marcação utilizada para criar páginas na Web.

Ele não é uma linguagem de programação, mas sim uma linguagem responsável pela estrutura do conteúdo.

Exemplos de formatação

A sigla W3C é responsável por padronizar a Web.
Para salvar um arquivo HTML, pressione Ctrl + S.

Um exemplo de código HTML é: <p>Olá Mundo</p>

Saída do programa: Olá Mundo

A variável x representa um valor numérico.

Texto sobrescrito: x2
Texto subscrito: H2O
